Getting Started

Create a vritual environment.

python -m env react-env

Then, activate the environment.

source react-env/bin/activate

Now install fastapi, uvicorn, sqlalchemy

pip install fastapi uvicorn sqlalchemy

We would want to create two folders, one for fastapi and the other for react . Inside the fastapi folder, let us create three files, namely, database.py , models.py and main.py .

In the file database.py , add the following lines of code.

from sqlalchemy import create_engine from sqlalchemy.orm import sessionmaker from sqlalchemy.ext.declarative import declarative_base # just a database name URL_DATABASE = "sqlite:///./players.db" engine = create_engine(URL_DATABASE, connect_args={"check_same_thread": False}) SessionLocal = session_maker(autocommit=False, autoflush=False, bind=engine) Base = declarative_base()

In the models.py file, add the following lines of code. This code creates a basic table with a few columns.

from database import Base from sqlalchemy import Column, Integer, Float, String, Boolean # let us create a db table that inherits from Base class Transfer(Base): __tablename__ = "transfers" #let us create columns now id = Column(Integer, primary_key=True, index=True) amount = Column(Float) category = Column(String) description = Column(String) is_income = Column(Boolean) date = Column(String)
